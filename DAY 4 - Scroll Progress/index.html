HTML

HTML is the standard markup language used to structure web pages. It organizes content with elements like headers, paragraphs, links, and lists.

CSS

CSS controls the visual style of HTML documents. It handles colors, fonts, layouts, and responsive design through rules and selectors.

JavaScript

JavaScript adds interactivity to web pages. It handles events, updates content, and integrates with APIs to make dynamic websites.

PHP

PHP is a server-side language used to generate dynamic content, work with databases, and manage backend logic in web development.

SQL

SQL is a language for interacting with databases. It is used to query, update, and manage structured data in relational database systems.
